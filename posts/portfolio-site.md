## 概要
Reactの学習のため、  
自己スキル数値化サイトを作成。  
1ページのみのシンプルなもの。  
GitHub　API通信を行い、  
個人で、どのプログラミング言語の  
開発をしてきたかがかわかる。  
デプロイはNetlifyを利用。  
 
##  技術
・React  
・React Hooks   
・react-circle  
・axios（HTTP通信）   
・Github API  
・Netlify

##  機能  
Githun APIと連携し、プログラミングごとの  
RepositoryをReact-circleを利用して表示
